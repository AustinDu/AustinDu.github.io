Y-Type





A project of a clone of the game Z-Type, made by Dominic Szablewski. This is a typing game, where objects with words fall from the top of the screen and the user has to destroy the objects by typing its associated word before it reaches the ship.

Done in Java, in 2014.

An original project of a game of tower defense. The user is able to place towers around the map to prevent the incoming waves of enemies from reaching the base. Money accumulates from enemy deaths, and more towers can be bought.
