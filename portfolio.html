MHacks 6 - Bloated
This was a project done at MHacks 6 in a team of four over 36 hours.
The web app simulated an interactive map of imports and exports using Bloomberg's API.
I worked on most of the front-end implementation and UI/UX design with HTML/CSS/Javascript.
Jquery and a d3 library was used to help animate the simulation.
Hack The North 2015 - Noflix
This was a project done at Hack The North 2015 in a team of four over 36 hours.
The app allowed the user to specify price and location radius before ordering an uber for the user to a random venue.
I worked on all of the front-end implementation and UI/UX design with Swift.
The Uber API, Yelp API, Eventbrite API, AlamoFire and SwiftyJSON were used to create this app.
